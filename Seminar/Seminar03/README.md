## For
```c++
for (<инициализация>; <условие>; <стъпка>) {
    <тяло>
}
```
## While
```c++
while (<условие>) {
    <тяло>
}
```
## Do while
```c++
do {
    <тяло>
} while (<условие>);
```

### 4 Задача
